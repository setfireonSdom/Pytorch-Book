# Code Directory Guide

The files in this directory are arranged from fundamentals to fuller project workflows:

1. `01_tensor_basics.py`: tensors, shapes, and basic reshaping operations
2. `02_autograd_basics.py`: gradients and backpropagation
3. `03_linear_regression.py`: the standard training loop in its simplest useful form
4. `04_custom_dataset.py`: how `Dataset` and `DataLoader` organize data
5. `05_mlp_classifier.py`: a first complete image classification baseline
6. `06_cnn_classifier.py`: moving from MLPs to CNNs
7. `07_transfer_learning.py`: a lightweight transfer learning template
8. `08_project_template.py`: a generic starting point for your own task
9. `09_tabular_binary_classification.py`: a fuller tabular binary classification project
10. `10_image_classification_project.py`: a fuller image classification workflow
11. `11_checkpoint_inference.py`: loading a saved checkpoint and running inference

The recommended study order is the file order.
